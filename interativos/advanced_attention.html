🧮 Detalhes Matemáticos

Fórmulas Fundamentais

Attention Score:
Attention(Q, K, V) = softmax(QK^T / √d_k)V
Multi-Head Attention:
MultiHead(Q, K, V) = Concat(head₁, ..., head_h)W^O
onde head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)
Scaled Dot-Product:
score_ij = (q_i · k_j) / √d_k
α_ij = exp(score_ij) / Σ_k exp(score_ik)

Complexidade Computacional

  • Tempo: O(n² × d) onde n é o comprimento da sequência
  • Espaço: O(n² + n × d) para armazenar scores e embeddings
  • Parâmetros: 4 × d × d_model (para Q, K, V, O)

🦀 Código Rust
fn scaled_dot_product_attention(
    q: &Tensor, // [batch, seq_len, d_k]
    k: &Tensor, // [batch, seq_len, d_k]
    v: &Tensor, // [batch, seq_len, d_v]
    mask: Option<&Tensor>,
    temperature: f32
) -> Tensor {
    let d_k = q.size(-1) as f32;
    let scores = q.matmul(&k.transpose(-2, -1)) / (d_k.sqrt() * temperature);
    
    let scores = if let Some(mask) = mask {
        scores.masked_fill(&mask, f32::NEG_INFINITY)
    } else { scores };
    
    let attention_weights = scores.softmax(-1);
    attention_weights.matmul(v)
}
